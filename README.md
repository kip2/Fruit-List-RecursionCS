# おまけ

プロジェクトのセットアップ&ビルドにシェルスクリプトを使用

## setup.sh

以下のコマンドで起動する

```shell
. setup.sh
```

作成するプロジェクトネームを入力する

プロジェクトネームでディレクトリが作成される

## run.sh

作成したプロジェクトのディレクトリに移動して使用する

```shell
# ディレクトリの移動
cd [project-name]

# run.shを起動
. run.sh
```

すると、自動でビルドが行われる

ビルドが成功したら、以下のURLで確認が可能

http://localhost:3000
